Release Process
===========================================================

## Update OSH Core release version

in osh-core folder:

```bash
mvn release:update-versions -DautoVersionSubmodules=true
```

## Create Tag on GitHub

Push latest changes + create new tag


## Deploy to Maven repo

```bash
mvn deploy -
```

## build osh-base distro
